Detailed Description

Behavior Tree framework: nodes, blackboard, and execution context.

Header-only BT framework with polymorphic nodes for runtime composition. Composite nodes (Sequence, Selector, Parallel), decorator nodes (Inverter, Repeater), and leaf nodes (Condition, Action) form a tree that is ticked each AI update to drive entity behavior.
