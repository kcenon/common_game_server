Detailed Description

template<typename T, typename E = Error>
class cgs::Result< T, E >

Result type for explicit error propagation.

Replaces exceptions with a type-safe, explicit error handling pattern. Every function that can fail returns Result<T, E> instead of throwing.

Template Parameters
TThe success value type.
EThe error type (defaults to cgs::Error).

Example:

auto result = parseConfig("server.yaml");
if (result.hasValue()) {
auto config = result.value();
} else {
log(result.error().message);
}
