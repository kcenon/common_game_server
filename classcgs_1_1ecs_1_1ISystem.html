Detailed Description

Abstract base class for ECS systems.

Concrete systems override Execute() and optionally GetStage() and GetAccessInfo(). The scheduler owns and manages the lifecycle of registered systems.

Return the component access patterns for this system.

Override to declare which component types the system reads and writes. Systems with declared, non-conflicting access patterns may be executed in parallel by the scheduler.

Default: empty (undeclared — always runs sequentially).
