Detailed Description

YAML-based configuration manager providing typed access to config values.

Supports loading from file, dotted-key access (e.g., "server.port"), setting values at runtime, and registering callbacks for change notification.

Internally flattens the YAML tree into a key-value map to avoid yaml-cpp reference-semantic pitfalls.
