Detailed Description

Game-specific job scheduler wrapping kcenon's thread_system.

Provides priority-based scheduling, dependency chaining, and tick-based recurring jobs for the game loop. Uses PIMPL to hide thread_system details from the public API, preventing header dependency leakage.

Example:

GameJobScheduler scheduler(4);
auto id = scheduler.schedule([] { computeAI(); }, JobPriority::High);
scheduler.wait(id.value());
