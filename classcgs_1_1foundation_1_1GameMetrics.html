Detailed Description

Central metrics facade providing counters, gauges, histograms, trace spans, health status, and Prometheus text-format export.

Thread-safe: counters and gauges use atomic operations; histograms and health maps are mutex-protected. The class is non-copyable but movable (PIMPL pattern).

Example:

auto& metrics = GameMetrics::instance();
metrics.incrementCounter("cgs_messages_total");
metrics.setGauge("cgs_ccu", 1500.0);
metrics.registerHistogram("cgs_tick_ms", HistogramBuckets::defaultLatency());
metrics.recordHistogram("cgs_tick_ms", 12.5);
std::string prom = metrics.scrape(); // Prometheus text format
