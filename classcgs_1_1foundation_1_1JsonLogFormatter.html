Detailed Description

Stateless JSON log formatter producing structured log lines.

Output format (ELK/Loki compatible):

{"timestamp":"2026-02-14T12:00:00.000Z","level":"INFO",
"category":"Core","correlation_id":"uuid","message":"..."}
