Detailed Description

template<typename... Args>
class cgs::foundation::Signal< Args >

Thread-safe signal (observer pattern) that dispatches events to registered callbacks.

Template Parameters
ArgsThe argument types passed to each slot when the signal fires.

Example:

Signal<SessionId> onConnected;
auto id = onConnected.connect([](SessionId sid) {
std::cout << "session " << sid.value() << " connected\n";
});
onConnected.emit(SessionId(42));
onConnected.disconnect(id);

◆ emit()

template<typename... Args>
void cgs::foundation::Signal< Args >::emit ( Args... args) const
inline

Fire the signal, invoking every registered slot with the given args.

Slots are invoked under a shared (reader) lock so emit() calls from multiple threads can overlap. However, each slot invocation itself is not protected — slots must be internally thread-safe if shared state is accessed.
