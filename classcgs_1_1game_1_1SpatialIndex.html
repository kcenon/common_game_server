Detailed Description

Grid-based spatial index for efficient entity queries.

Entities are placed into cells based on their X/Z world position. The grid is sparse (only occupied cells are stored), making it memory-efficient for large, mostly-empty worlds.

Thread safety: None. External synchronization is required if accessed from multiple threads.
