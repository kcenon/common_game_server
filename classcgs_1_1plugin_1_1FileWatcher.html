Detailed Description

Polling-based file change detector.

Watches shared library files by periodically checking their last-write timestamps. Changes within a configurable debounce window are coalesced into a single callback invocation.

Usage:

FileWatcher watcher;
watcher.SetCallback([](const std::filesystem::path& p) {
std::cout << "Modified: " << p << "\n";
});
watcher.Watch("/path/to/plugin.so");
watcher.Poll(); // Call periodically.
