Detailed Description

Abstract base class for game server plugins.

Every plugin — whether loaded dynamically (.so/.dll) or registered statically — must implement this interface. The PluginManager drives the lifecycle callbacks in order:

OnLoad() → OnInit() → OnUpdate()* → OnShutdown() → OnUnload()
