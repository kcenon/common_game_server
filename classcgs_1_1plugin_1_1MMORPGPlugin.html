Detailed Description

Reference MMORPG plugin that integrates all game systems.

System execution order per tick (managed by SystemScheduler): PreUpdate: WorldSystem (spatial index synchronization) Update: ObjectUpdateSystem, CombatSystem, AISystem PostUpdate: QuestSystem, InventorySystem
