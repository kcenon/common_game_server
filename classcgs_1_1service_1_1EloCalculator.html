Detailed Description

Static utility class for ELO/MMR rating calculations.

Uses the standard Elo formula: E(A) = 1 / (1 + 10^((R_B - R_A) / 400))

K-factor determines how much a single game can change a rating. Typical values: 32 (new players), 24 (established), 16 (masters).
