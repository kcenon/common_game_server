Detailed Description

Opcode-range-based routing table.

Example:

RouteTable routes;
routes.addRoute(0x0100, 0x01FF, "game");
routes.addRoute(0x0200, 0x02FF, "lobby");
auto match = routes.resolve(0x0150);
// match->service == "game"
