Detailed Description

Circuit breaker state machine for protecting inter-service calls.

Usage:

ServiceCircuitBreaker cb(CircuitBreakerConfig{.name = "auth"});
if (cb.allowRequest()) {
auto result = callAuthService();
if (result.hasValue()) {
cb.recordSuccess();
} else {
cb.recordFailure();
}
} else {
// Circuit is open, fail fast
}
