Detailed Description

Installs SIGINT and SIGTERM handlers and exposes a shutdown flag.

Only one SignalHandler instance should exist per process. The handler writes to a static atomic flag in an async-signal-safe manner (relaxed store on a lock-free atomic).

After shutdown is requested, the original default handlers are restored so that a second signal terminates the process immediately.
