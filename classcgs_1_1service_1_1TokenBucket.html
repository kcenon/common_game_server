Detailed Description

Token bucket rate limiter for per-key message throttling.

Each key (typically a session or IP) maintains a bucket that fills at a constant rate and can burst up to a configured capacity.

Example:

TokenBucket bucket(100, 50); // 100 burst, 50 tokens/sec refill
if (bucket.consume("session-1")) {
// Message allowed
}
