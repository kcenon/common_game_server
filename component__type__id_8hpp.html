Detailed Description

RTTI-free compile-time component type identification.

Each distinct component type T receives a unique integer ID the first time ComponentType<T>::Id() is called. The IDs are generated via an atomic counter, so they are safe to use from multiple threads although the exact numeric values are not deterministic across runs.
