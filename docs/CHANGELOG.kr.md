## 형식

본 프로젝트는 [Keep a Changelog 1.1.0](https://keepachangelog.com/en/1.1.0/)
과 [Semantic Versioning 2.0.0](https://semver.org/spec/v2.0.0.html)를 따릅니다.

섹션:

- **Added** — 새 기능
- **Changed** — 기존 기능의 변경
- **Deprecated** — 곧 제거될 기능
- **Removed** — 제거된 기능
- **Fixed** — 버그 수정
- **Security** — 취약점 수정
