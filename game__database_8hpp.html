Detailed Description

GameDatabase wrapping kcenon database_system for game-specific database access with connection pooling and RAII transactions.

Provides synchronous and asynchronous query execution, prepared statements, transaction support with RAII guards, and a configurable connection pool. Part of the Database System Adapter (SDS-MOD-005).
