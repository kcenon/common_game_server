Detailed Description

GameNetworkManager wrapping kcenon network_system for game-specific TCP/UDP/WebSocket communication.

Provides protocol-agnostic server management with message framing, session tracking, and opcode-based dispatch. Part of the Network System Adapter (SDS-MOD-004).
