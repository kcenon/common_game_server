Detailed Description

Map instance lifecycle management for the Game Server.

MapInstanceManager tracks logical map instances, their states, and player counts. It does not own ECS entities; the WorldSystem manages the spatial data. This class provides service-level orchestration: instance creation, draining, and shutdown.
