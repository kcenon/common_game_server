enum class cgs::plugin::PluginState : uint8_t
strong

Plugin lifecycle states.

State machine: Unloaded ──LoadPlugin()──→ Loaded ──InitializeAll()──→ Initialized ──→ Active ↑ │ └────────────────── UnloadPlugin() / ShutdownAll() ───────────────────┘ ↓ (on error) Error

Enumerator
Unloaded 

Not loaded; initial state.

Loaded 

Shared library opened, IPlugin created, OnLoad() succeeded.

Initialized 

OnInit() succeeded.

Active 

Running; receives OnUpdate() calls.

ShuttingDown 

OnShutdown() in progress.

Error 

An error occurred during a lifecycle transition.
