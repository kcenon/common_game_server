Detailed Description

Macros for exporting and registering plugins.

Dynamic plugins use CGS_PLUGIN_EXPORT to generate the C entry point that PluginManager calls via dlsym/GetProcAddress.

Static plugins use CGS_PLUGIN_REGISTER to add themselves to the compile-time registry so that PluginManager can discover them without dynamic linking.
