Detailed Description

Compact entity handle: 24-bit index + 8-bit version packed into 32 bits.

The version field prevents the ABA problem when entity IDs are recycled. EntityManager (Issue #10) will manage creation / destruction; this header only defines the value type so that ComponentStorage can use it.
