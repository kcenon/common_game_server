Detailed Description

Version constraint parsing and matching for plugin dependencies.

Supports version constraint operators: >=, >, <=, <, ==, ~= (compatible release). Dependency strings combine a plugin name with optional constraints: "PluginName" — any version "PluginName>=1.0.0" — version 1.0.0 or newer "PluginName~=1.5" — compatible release (>=1.5.0, <2.0.0)
