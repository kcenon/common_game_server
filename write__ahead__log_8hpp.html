Detailed Description

Write-Ahead Log for crash-safe player data persistence.

Records player state mutations as append-only binary entries with CRC32 integrity checksums. On crash recovery, entries are replayed to restore the last consistent state.

Part of SRS-NFR-013 (zero data loss on crash).
